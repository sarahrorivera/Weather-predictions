## Weather-predictions

# Overview

This project retrieves and analyzes historical weather data for Atlanta, Georgia on November 28th across multiple years (2020â€“2024). Using the Open-Meteo Archive API ,the program collects temperature, wind speed, and precipitation data, processes it with pandas, and outputs yearly summaries along with a five-year statistical average.


# Features 
- Fetch hourly temperature data from the Open-Meteo API
- Process and organize data into Pandas DataFrames for easy analysis
- Store and manage weather data in a database 
- Modular design with dedicated files for database logic, data handling, and app execution.
- Basic test script to verify setup
